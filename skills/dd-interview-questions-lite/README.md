# dd-interview-questions-lite

Public-safe lightweight due-diligence interview question generator.

## What It Does

This skill helps users prepare role-based interview questions from project materials.

It can generate:

- Project snapshot
- Key hypotheses to verify
- Role-based interview plan
- Questions with evidence requests
- Follow-up questions
- Cross-check questions
- Missing information list
- Suggested interview sequence

## What It Does Not Include

This public version does not include:

- Private diligence playbooks
- Proprietary question banks
- Internal file paths or automation scripts
- Confidential project examples
- Full production prompt chains

## Usage

```text
Use dd-interview-questions-lite to prepare interview questions for this project:

[paste project material]
```

You can also specify roles:

```text
Prepare questions for CEO, CTO, and finance lead.
```

## Install

Copy this folder into your skills directory:

```bash
~/.codex/skills/dd-interview-questions-lite
```

or:

```bash
~/.claude/skills/dd-interview-questions-lite
```

Restart your agent after installation.
