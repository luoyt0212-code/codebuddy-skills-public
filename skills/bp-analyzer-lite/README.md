# bp-analyzer-lite

Public-safe lightweight BP analysis skill for first-pass review.

## What It Does

This skill helps users review a business plan or pitch deck and produce:

- Executive summary
- Business model review
- Market and timing assessment
- Product and technology assessment
- Traction review
- Competition review
- Risk summary
- Follow-up questions
- Missing information list
- First-pass conclusion

## What It Does Not Include

This public version does not include:

- Private PE methodology
- Proprietary sector question banks
- Internal investment scoring systems
- Confidential client or project examples
- Full production prompt chains

## Usage

```text
Use bp-analyzer-lite to review this BP:

[paste project material]
```

## Install

Copy this folder into your skills directory:

```bash
~/.codex/skills/bp-analyzer-lite
```

or:

```bash
~/.claude/skills/bp-analyzer-lite
```

Restart your agent after installation.
