# CodeBuddy Skills Public

Public, shareable versions of selected CodeBuddy skills.

This repository intentionally contains only public-safe skill instructions. Private investment frameworks, proprietary prompt chains, internal operating methods, client information, and high-value question banks are not included here.

## Available Skills

### bp-analyzer-lite

A lightweight BP and pitch deck analysis skill for first-pass project review.

Path:

```text
skills/bp-analyzer-lite/SKILL.md
```

### dd-interview-questions-lite

A lightweight due-diligence interview preparation skill for drafting role-based interview questions.

Path:

```text
skills/dd-interview-questions-lite/SKILL.md
```

## Install

Clone this repository into your skills directory:

```bash
git clone https://github.com/luoyt0212-code/codebuddy-skills-public.git ~/.codex/skills/codebuddy-skills-public
```

Or for Claude Code:

```bash
git clone https://github.com/luoyt0212-code/codebuddy-skills-public.git ~/.claude/skills/codebuddy-skills-public
```

If your agent expects each skill to be a direct child of the skills directory, copy the specific skill folder:

```bash
mkdir -p ~/.codex/skills
cp -R ~/.codex/skills/codebuddy-skills-public/skills/bp-analyzer-lite ~/.codex/skills/bp-analyzer-lite
```

## Notes

The lite skills are designed for public sharing and learning. They are not the same as the author's private production skills.
